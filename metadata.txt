about=GeoClassify is a scientific raster classification toolkit for QGIS 4. It reclassifies raster data using thresholds accepted in the literature. Supported data types: Vegetation indices (NDVI, NDWI, SAVI, EVI, NDMI, NBR, BSI), Topographic data (Elevation, Slope, Aspect), Climate data (Annual Temperature, Annual Precipitation), Land use / land cover (LULC), Erosion Risk. Key features: Preview and edit classification ranges before processing, Create/save/import/export user-defined classifications, CRS-aware area calculation (projected and geographic/WGS84), Automatic QML style application, CSV export of per-class area statistics, Batch processing (reclassify an entire folder or selected files in one click).
